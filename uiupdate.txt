# ReCost UI Update — Plan & Prompts

## The Problem

The AST engine (Phases 1-3) produces rich data: 7 frequency classifications, per-method pricing, provider metadata, cross-file tracing, streaming flags, batch/cache capability markers, and calibrated confidence scores. But the pipeline has two choke points that throw most of this away:

1. **`astMatchToApiCallInput()` in workspace-scanner.ts** — collapses 7 frequency classes to binary "per-request"/"daily", drops provider, batchCapable, cacheCapable, streaming, isMiddleware, crossFile, sourceFile
2. **`estimateLocalMonthlyCost()` in webview-provider.ts** — uses a hardcoded `LOCAL_PRICING` flat-rate table instead of the fingerprint registry's per-method pricing

The UI also has display gaps:
- Frequency is never shown on endpoints
- Confidence is only shown for AI suggestions, not local-rule or remote
- Cost model type (per_token, per_transaction, free) is never shown
- No cross-file origin info
- No streaming/batch/cache capability indicators
- Provider display is plain text with no color coding

## Prompt Order

Do these in sequence. Each builds on the previous.

  import { lookupMethod } from "./fingerprints/registry";

  function estimateLocalMonthlyCost(
    provider: string, 
    callsPerDay: number,
    methodSignature?: string
  ): number {
    if (methodSignature) {
      const entry = lookupMethod(provider, methodSignature);
      if (entry) {
        // compute based on costModel
        if (entry.costModel === "free") return 0;
        if (entry.costModel === "per_token") {
          const inputTokens = entry.defaultInputTokens ?? 500;
          const outputTokens = entry.defaultOutputTokens ?? 200;
          const inputCost = (inputTokens / 1_000_000) * (entry.inputPricePer1M ?? 0);
          const outputCost = (outputTokens / 1_000_000) * (entry.outputPricePer1M ?? 0);
          return Math.round((inputCost + outputCost) * callsPerDay * 30 * 100) / 100;
        }
        if (entry.costModel === "per_transaction") {
          const txValue = entry.defaultTransactionUsd ?? 50;
          const fee = (entry.fixedFee ?? 0) + txValue * (entry.percentageFee ?? 0);
          return Math.round(fee * callsPerDay * 30 * 100) / 100;
        }
        if (entry.costModel === "per_request") {
          return Math.round((entry.fixedFee ?? entry.perRequestCostUsd ?? 0.0001) * callsPerDay * 30 * 100) / 100;
        }
      }
    }
    // fallback to old flat rate
    const perCall = LOCAL_PRICING[provider] ?? DEFAULT_PER_CALL_COST;
    return Math.round(callsPerDay * perCall * 30 * 100) / 100;
  }
